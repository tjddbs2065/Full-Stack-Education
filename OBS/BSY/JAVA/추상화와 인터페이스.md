##### 추상 클래스
- `abstract` 키워드, 직접 객체 생성 불가
- 추상 메서드(구현 X) 포함 가능
- 자식 클래스가 반드시 재정의

##### 인터페이스
- 모든 메서드: `public abstract` 기본
- 모든 변수: `public static final` 기본
- 다중 구현 가능, 다중 상속 대체
- 특정 행위/능력을 강제하는 규약 제공

##### UML 표기
- `+`: public / `-`: private / `#`: protected / `~`: default
- is-a: 상속 관계 / has-a: 포함 관계
