# JavaScript 비동기 처리

- [[싱글 스레드]] 언어이기 때문에, 한 번에 하나의 작업만 처리 가능

2. [[Promise]]

```javascript
async function fetchData() {
  const response = await fetch('https://example.com');
  const data = await response.json();
  console.log(data);
}
```
