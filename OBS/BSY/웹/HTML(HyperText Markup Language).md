##### HTML이란?
- 웹페이지의 뼈대(구조)를 만드는 마크업 언어
- 브라우저가 해석하여 화면에 출력한다.

##### 기본 구조
```html
<!DOCTYPE html>
<html>
  <head>
    <title>웹페이지 제목</title>
  </head>
  <body>
    <h1>안녕하세요</h1>
    <p>이것은 HTML 문서입니다.</p>
    <a href="https://example.com">링크</a>
  </body>
</html>
```
