##### 요구사항
- FAT32 파일시스템의 분석
	- USB 파일 및 폴더 구조 트리 형태로 조회
	- 파일 export
	- 삭제된 파일 구분하여 조회
	- 삭제된 파일 추출


##### 사용할 수 있는 디자인 패턴 조합
```
FileSystem (abstract)
 └─ Fat32FileSystem
     - parseBootSector()
     - createRootDirectory()

IFileSystemEntry (component)
 ├─ Directory (composite)
 └─ File (leaf)

ClusterIterator
DirectoryIterator

FileExtractor (template)
 ├─ RawFileExtractor
 └─ CompressedFileExtractor

ProgressObserver (observer)
 ├─ ConsoleLogger
 └─ GUIUpdater
```
