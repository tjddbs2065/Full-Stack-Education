##### 요구사항
- FAT32 파일시스템 유틸(사용자)
	- 파일시스템 로드
	- 파일과 폴더 구조 획득
	- 트리 형태로 변환
	- 선택한 파일 추출 및 저장
	- 삭제된 파일은 구분


##### 사용할 수 있는 디자인 패턴 조합
```
FileSystem (abstract)
 └─ Fat32FileSystem
     - parseBootSector()
     - createRootDirectory()

IFileSystemEntry (component)
 ├─ Directory (composite)
 └─ File (leaf)

ClusterIterator
DirectoryIterator

FileExtractor (template)
 ├─ RawFileExtractor
 └─ CompressedFileExtractor

ProgressObserver (observer)
 ├─ ConsoleLogger
 └─ GUIUpdater
```
